Create a News Link that can be Shared on Facebook Canada

Insert a link to a news story you want to share on Facebook Canada. Click the 'Create Link' button. It creates a Bing search link that displays one result. Share the link on Facebook. The link will generate a Facebook preview for the Bing search result, which you can keep or delete. The link will not be blocked.
