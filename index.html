Create a News Link that can be Shared on Facebook Canada

  1. Insert a link to a news story you want to share on Facebook Canada.
  2. Click the Create Link button. The tool creates a Bing search link featuring the one result. A Bing link will not be blocked.
  3. Share the link on Facebook. (The link could generate a Facebook preview for the Bing search, which you can keep or cancel.)
